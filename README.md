# 多元選修課程成果 - 互動式網站

## 專案簡介
這是一個整合型的網頁互動專案，包含：
- 🐍 貪食蛇遊戲
- ✏️ 繪圖板
- 📝 留言板（需搭配 Node.js + MySQL）

## 技術棧
- HTML5 / CSS3 / JavaScript
- Canvas API
- Node.js + Express（後端，影片中展示）

## 如何執行（完整版）
1. 安裝 Node.js 套件：`npm install`
2. 設定資料庫連線（需學校環境）
3. 啟動伺服器：`node server.js`
4. 開啟瀏覽器：`http://localhost:3000`
